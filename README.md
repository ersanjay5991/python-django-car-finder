# Django projects 
### Prerequisites
```
python == 3.6

sudo apt install python3
sudp apt install python3-pip

Django == 2.0.7

pip install django 
```
### Installing
```
$ pip install -r requirements.txt
```
### Running
```
$ python manage.py makemigrations
$ python manage.py migrate
$ python manage.py runserver
```
